# # 🏛️ THE ETERNAL ARCHIVE: CLASS X-A

> "Sebuah ruang digital untuk mengabadikan jejak, tawa, dan cerita yang pernah kita lukis bersama di kelas X-A."

Selamat datang di repositori resmi **The Eternal Archive**. Proyek ini dibuat untuk memastikan bahwa setiap kenangan kita tidak hilang dimakan waktu, melainkan tersimpan rapi dalam bentuk kode dan visual.

---

## 🌐 Live Preview
Lihat website kami yang sudah mengudara di:
**[https://the-xa.me](https://the-xa.me)**

---

## ✨ Fitur Utama
* **Interactive 3D Book:** Menampilkan foto-foto anggota kelas dengan efek estetik.
* **Memory Timeline:** Alur perjalanan kelas dari awal hingga akhir semester.
* **Music Background:** Suasana yang mendukung untuk bernostalgia.
* **Responsive Design:** Bisa dibuka dengan nyaman lewat HP maupun Laptop.

## 🛠️ Teknologi yang Digunakan
* **HTML5** & **CSS3** (Struktur & Desain)
* **JavaScript** (Logika interaktif & Efek)
* **GitHub Pages** (Hosting gratis seumur hidup)
* **Custom Domain** (the-xa.me via Namecheap)

---

## 📂 Struktur File
- `index.html` - Halaman utama website.
- `music.mp3` - Musik latar kenangan.
- `CNAME` - Konfigurasi domain kustom.
- `img/` - Folder penyimpanan seluruh aset foto anggota dan momen.
